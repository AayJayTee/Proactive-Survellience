Project Overview

An intelligent surveillance system designed to transform raw video footage into actionable insights. It combines computer vision and AI to automatically detect events, identify anomalies, and generate human-readable summaries. The system enhances situational awareness by highlighting important activities and organizing them into a structured timeline. With built-in explainability, every alert is supported by visual and contextual evidence. It reduces manual monitoring effort and enables faster, more informed decision-making.

Action Recognition

The action recognition module analyzes video sequences to identify human activities such as running, fighting, or loitering. It uses spatio-temporal deep learning models to understand both motion and visual context across frames. By processing clips instead of individual images, it captures how actions evolve over time. The module outputs action labels along with confidence scores for reliability. These predictions form the foundation for higher-level analysis and summarization. It serves as the core behavioral understanding component of the system.

Anomaly Detection

The anomaly detection module identifies unusual or suspicious behavior by learning patterns of normal activity. It uses reconstruction-based models to detect deviations in video frames, flagging events that differ from expected behavior. High reconstruction error indicates potential anomalies, such as unexpected motion or irregular actions. This approach enables detection without requiring extensive labeled abnormal data. The system provides anomaly scores to quantify severity and trigger alerts. It plays a key role in proactive surveillance and risk detection.

Explainability

The explainability module ensures that every AI decision is transparent and interpretable. It uses techniques like Grad-CAM and activation maps to highlight the exact regions in the video that influenced a prediction. This allows users to visually understand why an action or anomaly was detected. By linking model outputs with clear visual evidence, it builds trust and reduces ambiguity. The module also aids in debugging and validating system behavior. It transforms the system from a black-box model into an accountable and reliable surveillance solution.
